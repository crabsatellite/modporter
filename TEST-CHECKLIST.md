# ModPorter — 游戏内测试清单

> **测试环境**: NeoForge 1.21.1 + Java 21，创造模式单人世界
> **JAR 安装**: 将对应 JAR 放入 `.minecraft/mods/` 目录
> **每项标记**: ✅ PASS / ❌ FAIL / ⏭️ N/A

---

## 1. Radios（无线电通讯方块）

**JAR**: `Radios-neoforge/build/libs/radios-1.2.0.jar`

### 准备
- `/gamemode creative`
- 按 E 打开物品栏，搜索 "Radio"

| # | 测试项 | 操作步骤 | 预期结果 | 结果 |
|---|--------|---------|---------|------|
| 1.1 | 物品存在 | 物品栏搜索 "radio" | 找到 "Band 5 Radio" 方块 | |
| 1.2 | 放置方块 | 将 Radio 放在地上 | 方块正常放置，贴图正常显示 | |
| 1.3 | 破坏方块 | 打掉 Radio | 正常破坏并可捡起 | |
| 1.4 | 发送消息 | 手持钻石，右键点击 Radio | 聊天栏出现 "Tuning..." 提示 | |
| 1.5 | 接收消息 | 空手右键点击同一个 Radio | 聊天栏显示已广播的消息 | |
| 1.6 | 远距离测试 | 在 ~300 格外放第2个 Radio，去远处接收 | 信号衰减或收不到 | |
| 1.7 | 存档持久化 | 放置 Radio 并发消息 → 退出 → 重进 | Radio 仍在，数据保留 | |
| 1.8 | 压力测试 | 快速连续右键 Radio 10次 | 不崩溃，无报错 | |
